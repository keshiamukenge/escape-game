VOICI LES REGLES


Vous êtes à l'intérieur d'une maison.
De nombreux indices sont dissimulés dans la pièce.
À l'aide de ces indices vous pourrez déchiffrer le code secret qui vous permettra d'accéder à l'étape suivante.
Ce code est à trouver dans un temps impartie.
Pour vous aider, des astuces apparaîtront au fur et à mesure du jeu.
À vous de jouer !
